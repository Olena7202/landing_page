# Bike Landing Page

A responsive landing page for a bike showcase with product highlights, feature sections, and a contact form.

## Demo

[Demo link](https://Olena7202.github.io/landing_page/)

## Design

The page includes:
- hero section with navigation
- bike comparison cards
- product details gallery
- contact section

## Technologies

- HTML5
- SCSS
- JavaScript
- Vite

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Olena7202/landing_page.git
cd landing_page
```

2. Install dependencies:

```bash
npm install
```

3. Run the project locally:

```bash
npm start
```

4. Build for production:

```bash
npm run build
```
